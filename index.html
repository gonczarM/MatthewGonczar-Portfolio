Allo

Allo may take longer to initially load due to Heroku's server sleep mode. Once the app loads up, create an an account and search for my username, gumballg, Send me a message and I'll try to reply back to you.

Allo is an instant translation messaging app. Each user can have their recieved messages translated to a prefered language. It completes these translations using IBMs Watson Language Translator API.

The back end was built using Node.js, express, and MongoDB. The front end was created using the ReactJS library.

Legend of Link

This was the first project I was able to build using only JavaScript. I grew up playing many iterations of Legend of Zelda and wanted to emulate it. This game lets you control and move Link around the map. You share the map with many ghosts that will attack when you cross their path. You can, however, try and defeat the enemies by swinging your sword at them.

Legend of Link was created using canvas, JavaScript, HTML, and CSS
