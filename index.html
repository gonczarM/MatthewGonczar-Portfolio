Allo

Allo is an instant translation messaging app that receives and immediately translates messages to the user’s preferred language. When both users are active in a conversation Allo takes advantage of socket.io, updating the conversation with sent messages instantly for both parties. Open the app to create an account and search for my username, “gumballg” send me a message and I will reply back to you. Occasionally there is a delay on the intial load so patience is appreciated.

Technologies

Node.js | Express | MongoDB | ReacJS | IBM Watson | socket.io

Legend of Link

Legend of Link was my first project built using only JavaScript. I grew up playing many iterations of Legend of Zelda and wanted to emulate it. This is a basic take of Link to the Past which lets you control and move Link around the map to dodge and attack enemies. Try to defeat all the enemies by swinging your sword at them and saving Hyrule from the evil Ganondorf.

Technologies

Canvas | JavaScript | HTML | CSS

Gregslist

Gregslist was created to be a more job oriented “craigslist”. This full-stack application was developed for users to create job postings, events, and upload their own résumé. Users can RSVP to events and connect with others by emailing through this application. Occasionally there is a delay on the intial load so patience is appreciated.
